# 店铺营销活动 H5 原型

## 📁 文件说明

- **list.html** - 活动列表页（首页）
- **create.html** - 创建活动页

## 🚀 使用方式

### 方式 1：直接打开
直接在浏览器中打开 `list.html` 即可预览。

### 方式 2：本地服务器（推荐）
```bash
# 使用 Python 启动本地服务器
cd h5-prototype
python3 -m http.server 8080

# 访问 http://localhost:8080/list.html
```

## 📱 页面功能

### 活动列表页 (list.html)
- 店铺选择器
- 空状态引导卡片（手动创建/智能生成）
- 活动筛选和搜索
- 活动卡片列表（审批状态、活动信息、操作按钮）
- 底部导航栏
- 活动统计弹窗
- 活动启/停切换

### 创建活动页 (create.html)
- 活动门店选择
- 活动名称输入
- 活动日期选择
- 适用业务选择（自提/小程序外卖）
- 活动人群选择（全部人群/仅会员 + 会员等级）
- 集点规则设置
- 商品范围选择
- 礼品设置（优惠券、有效期、数量等）
- 活动说明（折叠面板）
- 申请原因输入
- 提交审批

## 🎨 设计规范

- **主色调**: #2860FA (蓝色)
- **圆角**: 8-12px
- **字体**: 系统字体（Apple-System, PingFang SC）
- **适配**: 移动端最大宽度 450px

## 📝 注意事项

1. 两个页面通过链接相互跳转，建议放在一起
2. 所有样式和脚本都内嵌在 HTML 文件中，无需额外依赖
3. 数据为模拟数据，仅用于 UI 展示
